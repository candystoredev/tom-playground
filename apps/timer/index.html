Original Request

“Think of 12 ideas for single-page web apps for app.tomhoeck.com. Apps a guy like me would actually use — into guitars, garage gym, shooting, woodworking, poker, and gaming. Single HTML files, no dependencies, great on mobile.”

Among those 12, Garage Gym Timer was specified with:

  • HIIT: configurable work/rest/rounds
  • Tabata: 20s/10s/8 rounds default
  • EMOM: every minute on the minute
  • Custom named segments, any duration
  • Voice cues via Web Speech API
  • Preset saving and rep counter

Changes

  • Initial build — HIIT, Tabata, EMOM, custom, rep counter
